# docs 目录说明（当前结构）

本目录整理了 HealthTrack 项目的规范、报告、演示与辅助资料，按实际子目录分组如下：

## Requirement（需求与规范）
- `Requirement/project_deliverable_1.md`
- `Requirement/project_deliverable_2.md`
- `Requirement/project_deliverable_3.md`
- `Requirement/project_description.md`
- `Final-Report.md`（课程报告草稿）

## 总览与演示
- `code_overview.md`：架构/模块/API 概览
- `feature_manual_detailed.md`：全功能详解 + curl 示例
- `demo_playbook.md`：演示步骤

## Detail（提交辅助与运行资料）
- `Detail/api_reference.md`：REST API 参考
- `Detail/db_schema_detail.md`：表级约束/PK/FK/索引汇总
- `Detail/seed_data_overview.md`：造数覆盖与验证 SQL
- `Detail/run_and_troubleshoot.md`：运行说明与排障
- `Detail/screenshots_guide.md`：截图/打印命名与覆盖点建议
- `Detail/curl_samples.sh`：核心流程 curl 示例
- `Detail/feature_showcase.md`：功能截图占位与示例操作说明

若需快速了解如何运行或演示，可查看 `Detail/run_and_troubleshoot.md` 与 `demo_playbook.md`。
